## ğŸ“¦ Getting Started

### Prerequisites

- Node.js >= 22
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/Aimony/lumina.git

# Enter the directory
cd lumina

# Install dependencies
npm install
```

### Development

Start the local development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```
