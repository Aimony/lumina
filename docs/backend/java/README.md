# Java编程语言学习指南

欢迎来到Java编程语言的完整学习指南！本目录包含了Java编程的各个方面，从基础概念到高级特性，帮助您系统地学习和掌握Java编程。

## 目录结构

```
docs/
└── java/
    ├── basics/           # Java基础概念
    ├── oop/             # 面向对象编程
    ├── collections/     # 集合框架和泛型
    ├── concurrency/     # 并发编程
    └── advanced/        # 高级特性和性能优化
```

## 各章节概览

### 1. Java基础概念 (basics/)

- Java语言概述和特点
- 基本语法和数据类型
- 变量、运算符和控制语句
- 数组和字符串处理
- 异常处理机制
- Java内存模型和垃圾回收

### 2. Java面向对象编程 (oop/)

- 类和对象的概念
- 封装、继承、多态三大特性
- 接口和抽象类
- 内部类和匿名类
- Object类和设计模式基础
- 最佳实践和设计原则

### 3. Java集合框架和泛型 (collections/)

- Collection接口层次结构
- List、Set、Queue、Map详解
- 各种实现类的特点和使用场景
- 泛型编程详解
- 集合工具类和性能比较
- 并发集合类

### 4. Java并发编程 (concurrency/)

- 线程基础和创建方式
- 线程同步和锁机制
- 线程间通信
- 线程池详解
- 并发集合和原子类
- 并发工具类（CountDownLatch、CyclicBarrier、Semaphore等）
- CompletableFuture和异步编程

### 5. Java高级特性和性能优化 (advanced/)

- Java 8-17新特性详解
- Lambda表达式和Stream API
- 新的日期时间API
- 记录类、密封类、模式匹配等现代特性
- JVM性能调优
- 代码层面的性能优化
- 内存管理和泄漏防范
- 数据库访问优化
- 缓存策略
- 性能监控和分析

## 学习建议

1. **初学者**：建议从`basics/`开始，逐步学习到`oop/`
2. **中级开发者**：重点关注`collections/`和`concurrency/`章节
3. **高级开发者**：深入学习`advanced/`章节，掌握性能优化技巧
