# 知识图谱：连接你的思维孤岛

## 什么是知识图谱？

**知识图谱**本质上是一种基于图的数据结构，由**节点 (Note)** 和**边 (Link)** 组成。在我们的 **Lumina** 系统中，每一篇 Markdown 笔记就是一个节点，而笔记之间的引用（如 `[[Digital Garden]]`）则构成了边。

这种结构模仿了人类大脑的神经元连接方式。与传统的文件夹分类（树状结构）不同，知识图谱是网状的。

> [!TIP]
> 试着把鼠标悬停在这个链接上：[[Digital Garden]]。这就展示了我们最新的**智能悬浮卡**功能！

## 为什么它很重要？

1.  **发现隐性联系**：当你积累了大量笔记后，图谱可以帮助你发现两个看似无关的主题之间的联系。
2.  **上下文增强**：通过 [[Smart Connections]]，我们不仅能看到谁链接了当前文章，还能理解链接的上下文。
3.  **导航的自由**：你不再受限于线性的目录结构，而是可以在知识网络中自由穿梭。

## 技术实现

我们的图谱视图使用了 D3.js 引擎，通过力导向布局算法（Force-directed graph）来实时渲染。
当你点击“全局视图”时，你会看到一个壮观的星系。
