# 智能连接：上下文即是王道

## 告别盲目的跳转

在传统的维基系统中，点击一个链接往往是一次赌博——你不知道点击后会看到什么，也不知道是否值得打断当前的阅读流。

Lumina 引入了**智能连接**的概念，核心体现就是你现在正在使用的**悬浮预览卡片 (Hover Cards)**。

## 工作原理

当你创建一个指向 [[Knowledge Graph]] 的链接时，系统会在构建时建立索引。

1.  **预加载索引**：我们在应用启动时加载了一个轻量级的 `search-index.json`。
2.  **即时渲染**：当你悬停在 `[[Digital Garden]]` 上时，我们从内存中直接读取目标文章的摘要和元数据。
3.  **零延迟**：不需要额外的 API 请求，体验丝般顺滑。

## 下一步计划

我们计划引入 AI 辅助的连接推荐。想象一下，当你写下“第二大脑”时，系统自动提示你连接到 [[Knowledge Graph]] 或 [[Digital Garden]]，这将极大降低维护知识库的心智负担。
