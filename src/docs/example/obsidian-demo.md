# Obsidian Syntax Support Demo

This page demonstrates the support for generic Obsidian markdown syntax.

## Wikilinks

Support for `[[Internal Link]]` syntax.

- Link to Home: [[/]]
- Link to Games: [[/games]]
- Link with text: [[/games/tetris|Play Tetris]]
- Link to a file: [[/README|README]]
- Link to a file: [[/example/obsidian-demo2]]

## Highlights

Support for `==highlighted text==` syntax.

This is ==highlighted text== using Obsidian syntax.

## Callouts (GitHub Alerts)

Support for Callouts (mapped to GitHub Alerts).

> [!NOTE]
> This is a note callout.

> [!TIP]
> This is a tip callout.

> [!IMPORTANT]
> This is an important callout.

> [!WARNING]
> This is a warning callout.

> [!CAUTION]
> This is a caution callout.
