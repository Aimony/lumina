# Obsidian 语法扩展演示

本页面展示了已实现的 Obsidian 增强语法功能。

## 1. 双向链接 (Wikilinks)

使用 `[[文件名]]` 或 `[[文件名|显示文本]]` 进行快速跳转。

- **基础链接**: [[/docs/guide/intro]]
- **别名链接**: [[/|返回首页]]
- **嵌套路径**: [[/docs/example/obsidian-demo|查看另一个示例]]

## 2. 高亮 (Highlights)

使用 `==内容==` 来高亮文本。

- 这是一个 ==非常重要== 的标记。
- Obsidian 风格的高亮让内容更易于阅读。

## 3. 内部注释 (Comments)

使用 `%%内容%%` 进行内部备注，这些内容在渲染后的页面中是 **不可见** 的。

- 这里有一行注释：%% 你在页面上看不见我，只能在编辑器里看到 %%
- 以上空白处其实隐藏了信息。

## 4. 内容嵌入 (Embeds)

使用 `![[文件名]]` 嵌入图片或其他资源。

- **图片嵌入**: ![[https://vite.dev/logo.svg|Vite 图标]]

## 5. 标注 (Callouts)

支持 Obsidian 风格的标注块，并支持 **可折叠** 属性。

### 基础标注

> [!INFO] 信息提示
> 这是一个标准的信息标注块。

### 可折叠标注

> [!TIP]+ 默认展开的提示
> 使用 `[!TIP]+` 语法，这个块在页面加载时会自动展开。

> [!WARNING]- 默认启动折叠的警告
> 使用 `[!WARNING]-` 语法，用户需要点击标题才能查看其中的详细内容。

### 更多样式

> [!IMPORTANT] 重要事项
> 请确保在生产环境测试这些功能。

> [!CAUTION] 危险操作
> 这是一个警示块，用于提醒潜在风险。

> [!DANGER] 严重警告
> 这是最高级别的警告样式。
