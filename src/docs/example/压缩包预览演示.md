# 压缩包预览功能演示

这是一个演示页面,展示如何在 Markdown 文档中嵌入 ZIP 压缩包文件并实现在线预览。

## 功能介绍

压缩包预览功能允许你:

- ✅ 直接在浏览器中查看 ZIP 文件内容
- ✅ 浏览文件夹层级结构
- ✅ 查看每个文件的详细信息(名称、大小、修改时间)
- ✅ 实时搜索和过滤文件
- ✅ 下载单个文件或整个压缩包
- ✅ 支持亮色/暗色主题
- ✅ 响应式移动端适配

## 使用方法

在 Markdown 中,你可以使用 `<ArchiveFileCard>` 组件来嵌入压缩包文件:

```vue
<ArchiveFileCard src="/files/ZIP测试.zip" name="示例压缩包.zip" />
```

## 技术细节

### 核心技术

- **JSZip**: 用于在浏览器中解析和处理 ZIP 文件
- **Vue 3**: 组件化开发框架
- **TypeScript**: 类型安全

### 组件架构

1. **ArchiveFileCard**: 卡片组件,用于在 Markdown 中嵌入
2. **ArchiveViewer**: 预览组件,显示压缩包文件树
3. **FileTreeNode**: 文件树节点组件,递归渲染文件结构

### 支持的格式

- ✅ ZIP (完全支持)
- ❌ RAR (不支持,需要额外的库)
- ❌ 7Z (不支持,需要额外的库)
- ❌ 加密压缩包 (不支持)

## 注意事项

> [!IMPORTANT]
> 压缩包文件会在浏览器中完全加载到内存,因此:
>
> - 建议压缩包大小不超过 100MB
> - 大文件可能导致浏览器卡顿
> - 文件数量建议在 1000 个以内

> [!TIP]
> 如果需要预览大型压缩包或其他格式,建议:
>
> - 先下载到本地
> - 使用专业的解压软件查看
